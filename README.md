# MeatKit (OtherLoaderPatched branch)

Stores source code for the .unitypackage in Assets/Packages/Otherloader/

### How to setup environment for editing code
1. Remove H3VR_IMPORTED from ProjectSettings/ProjectSettings.asset
2. Remove all Otherloader files in Assets/Packages/
3. Open this project in your Unity 5.6.7 editor, set up MeatKit normally
4. Double click any .cs file in Unity, which will create a .sln file in the root
5. Open the .sln with your IDE of choice
6. Revert the removed files via git
